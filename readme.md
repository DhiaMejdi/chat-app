# Tornado Chat App

A simple real-time chat application built with **Tornado** and **MySQL**.

## Features

- Real-time messaging using WebSocket
- Multiple chat rooms
- User registration on the fly
- Stores messages in a MySQL database
- Frontend with HTML, CSS, and JavaScript

## Requirements

- Python 3.8+
- MySQL Server
- Python packages (install via `pip install -r requirements.txt`)
